INTRODUCTION                                  *vim-gfm-syntax-introduction*

*gfm-syntax* provides extra syntax highlight for GitHub Flavored Markdown.
GitHub adds their extra features to Markdown.  This plugin fills the gap
between normal Markdown and GitHub Flavored Markdown.

Ref: https://help.github.com/categories/writing-on-github/

This plugin is enabled automatically on "markdown" filetype.  For more
customization, please see README.

https://github.com/rhysd/vim-gfm-syntax#readme


==============================================================================
INSTALL                                            *vim-gfm-syntax-install*

If you don't use any plugin manager, copy all files in after/, plugin/ and
doc/ to your .vim directory.

If you use a plugin manager, please follow the instruction of it to install
|gfm-syntax|.

Below is an example for |neobundle.vim|.
>
    NeoBundle 'rhysd/vim-gfm-syntax'
<
